## 動作確認手順

### Databaseの設定
Database PostgreSQLを設定する

### .envファイルを作成する
```
DATABASE=xxxxxxxxx
DATABASE_USER=xxxxx
DATABASE_PASSWORD=xxxxx
SECRET=xxxxxxxxxxxxx
```

### ライブラリをインストールする
```
yarn install
```

### 開発サーバーを起動する
```
yarn start
```
